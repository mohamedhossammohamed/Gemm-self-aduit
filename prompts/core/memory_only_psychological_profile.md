# Gemini Persistent-Memory Self-Analysis Prompt

You are conducting a longitudinal behavioral and cognitive analysis of the user based STRICTLY AND EXCLUSIVELY on persistent memory and prior conversations stored internally within Gemini.

## Absolute Constraints

You MUST NOT use:
- internet data,
- world knowledge about demographics,
- generalized personality stereotypes,
- statistical assumptions,
- external retrieval systems,
- cultural profiling,
- inferred social-group priors.

Use ONLY:
- saved memory,
- conversational history,
- writing style,
- interaction patterns,
- repeated themes,
- prompt structures,
- emotional signals embedded in language,
- behavioral consistency across conversations.

If a conclusion cannot be supported from memory-derived evidence, explicitly state uncertainty rather than fabricating depth.

## Objective

Construct a systems-level model of the user across cognition, psychology, behavioral dynamics, identity construction, motivational architecture, operational strengths/weaknesses, and relationship to AI systems.

This is NOT therapy, roleplay, or motivational coaching.

## Required Methodology

For every major claim:
1. State the observation.
2. State the inferred pattern.
3. State confidence: low, moderate, or high.
4. Distinguish evidence, interpretation, and speculation.

## Required Analysis Areas

1. Cognitive Architecture
2. Psychological Dynamics
3. Communication Signature
4. Behavioral Patterns
5. AI Relationship Analysis
6. Risk & Failure Mode Analysis (only if evidence-supported)

## Output Format

Include sections, confidence levels, evidence examples, contradictions, unresolved ambiguities, strongest patterns, and weakest inferences.

Conclude with:
1. Core behavioral model
2. Likely future trajectory
3. Highest-leverage improvement area
4. Most dangerous cognitive trap
5. Most underutilized strength

Maintain a clinical, analytical, non-therapeutic tone.
