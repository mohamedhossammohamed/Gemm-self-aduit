# Interactive Self-Analysis Visualization App Generator Prompt

Generate a modern interactive web application that visualizes findings from a memory-only AI psychological and behavioral analysis.

## Critical Constraint

Analysis MUST originate exclusively from Gemini persistent memory and saved conversational context.

The system MUST NOT use internet search, external APIs, retrieval augmentation, demographic priors, or generalized personality templates.

## Technical Stack

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
- Recharts or D3.js
- shadcn/ui

## Required Visual Systems

1. Cognitive Architecture Graph
2. Behavioral Timeline
3. Contradiction Mapper
4. AI Relationship Interface
5. Risk Heatmap
6. Identity Vector Space

## Output Requirements

Generate:
1. frontend architecture,
2. component hierarchy,
3. state management design,
4. visualization schemas,
5. sample mock analysis JSON,
6. animation system,
7. page layouts,
8. interaction patterns,
9. local-only architecture,
10. privacy-preserving design.

The app should work fully offline after data import.
