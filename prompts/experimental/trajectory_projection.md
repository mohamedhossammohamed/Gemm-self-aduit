# Trajectory Projection

Experimental prompt for generating longitudinal future-direction hypotheses based on observed patterns, with explicit confidence levels and uncertainty acknowledgment.

## Absolute Constraints

You MUST NOT use:
- Internet data or external predictive models
- Statistical baselines from population data
- Generalized life-stage assumptions
- Demographic priors
- External trend analysis
- Astrological or pseudoscientific frameworks

Use ONLY:
- Saved persistent memory and conversational history
- Observed behavioral patterns and trajectories
- Stated values and intentions
- Historical decision-making consistency
- Cognitive and operational strengths/weaknesses
- Identified growth edges and failure modes
- Trajectory of recent changes (accelerating, stabilizing, reversing)

If sufficient evidence does not exist for a projection, explicitly state "insufficient evidence" rather than extrapolating.

## Objective

Based on strict evidence from memory-only analysis, generate hypothesis-level projections of the user's likely future directions across key life/operational domains over 6-month, 1-year, and 3-year horizons.

This is NOT fortune-telling, coaching, or prescriptive guidance. It is structured pattern extrapolation with marked uncertainty.

## Required Methodology

For each projected direction:
1. **Observed Foundation** - What pattern from memory supports this projection?
2. **Extrapolation Logic** - How does this pattern likely continue or evolve?
3. **Confidence Level** - low, moderate, or high
4. **Alternative Scenarios** - What could disrupt this trajectory?
5. **Evidence vs. Speculation** - Clearly separate ground truth from inference

## Projection Domains

1. **Career/Operational Trajectory** - Role evolution, scope expansion, domain shifts
2. **Cognitive Development** - Sophistication of thinking, tool mastery, analytical depth
3. **Behavioral Stability** - Consistency vs. volatility in core patterns
4. **Risk Exposure** - Likelihood of encountering identified failure modes
5. **Relationship to AI Systems** - Evolution of usage modes and dependency
6. **Identity Evolution** - Coherence or transformation of core narratives
7. **Value-Action Alignment** - Convergence or divergence between stated and enacted values

## Time Horizons

Generate projections for:
- **6-Month Trajectory** (highest evidence reliability)
- **1-Year Trajectory** (moderate evidence reliability)
- **3-Year Trajectory** (lowest evidence reliability; mark explicitly)

## Output Format

Organize as:
1. **Executive Summary** - Core projected pattern across domains
2. **Detailed Domain Projections** - One section per domain with evidence, logic, confidence, and alternatives
3. **Cross-Domain Synergies** - How trajectories in one domain may reinforce or conflict with others
4. **Key Inflection Points** - Decision moments or external triggers that could alter trajectories
5. **Highest-Confidence Projections** - Where evidence is strongest
6. **Lowest-Confidence Projections** - Where evidence is weakest or contradictory
7. **Unresolved Uncertainties** - What would be needed to increase confidence?
8. **Counter-Evidence** - What patterns in memory contradict the main projections?

## Validation Notes

- Flag any projections that assume personality traits should remain static (they don't necessarily)
- Flag any projections based on "ought" rather than observed "is"
- Mark explicitly where you're extrapolating beyond the available data
- Acknowledge the irreducible uncertainty in long-term human behavior prediction
